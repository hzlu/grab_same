# hello_same

A command line client for [SAME](https://same.com/) app.

## Usage

search channel

```bash
channel -s <name>
```

view channel info or user profile

```bash
channel -i <id>
user -i <id>
```

view channel/user SAME posts

```bash
channel -c <id>
user -c <id>
```

show more posts(next page)

```bash
n/next/m/more
```

backup channel/user media resources

```bash
channel -b <id>
user -b <id>
```

help and quit

```bash
q/quit
h/help
```
